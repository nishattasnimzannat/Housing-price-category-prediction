🏠 Housing Price Category Prediction using Machine Learning & Neural Networks
Overview

This project was developed as part of the CSE422: Artificial Intelligence course.

The objective is to predict the price category of residential properties (Low, Medium, or High) based on various housing features. Multiple supervised learning algorithms and an Artificial Neural Network (ANN) were implemented and compared to evaluate their performance.

Features
Data preprocessing and cleaning
Exploratory Data Analysis (EDA)
Missing value handling
Label encoding for categorical features
Feature scaling using StandardScaler
Train-test split
Model training and comparison
Performance evaluation using multiple metrics
Unsupervised clustering with K-Means
Dataset

The dataset contains various housing attributes such as:

Property Size (sq ft)
Number of Bedrooms
Building Age
Distance to City Center
Location
Security Level
Parking Availability
Other property-related features

Target Variable

Low
Medium
High
Machine Learning Models

The project compares the following models:

Artificial Neural Network (ANN)
K-Nearest Neighbors (KNN)
Decision Tree
Logistic Regression
K-Means Clustering (for unsupervised analysis)

Evaluation Metrics

The models were evaluated using:

Accuracy
Precision
Recall
F1-Score
Confusion Matrix
ROC Curve & AUC Score
